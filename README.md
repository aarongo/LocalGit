#Gitlab Install For Centos 6.4

    操作系统:Centos6.4 64位
    Web服务器: apache
    数据库版本:MySQL 5.5.44
    GitLab 7.11.4
    GitLab Shell 2.6.3
    GitLab API V3
    Ruby 2.1.0p0
    Rails 4.1.9
##Start the instance
    service gitlab start|restart|status
    service httpd start|restart|status
